Description

For the first every sound class we had, Me with very little expierence with real sound design decided to make a super simple digital synthesizer that anyone could use by just seeing what happends on the screen. So the performer has a midi controller and the input is visualised directly on the laptop screen. For the observers there is a visualiser behind the performer which is a audioreactive vjshow.
