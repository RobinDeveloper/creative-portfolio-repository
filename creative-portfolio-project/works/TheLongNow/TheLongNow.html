Description

For this project we were tasked to make work inspired by The Long Now foundation. I started thinking what is something that takes forever. Encryption. So I ended up creating a website with a encrypted file on it with a method that would take a billion billion years to bruteforce. And the website tries to do that. The file has a little story inside about The Long Now and a lon/lat value. The locationg that points too is where a piece of silver is buried. With on it. The encryption key.
