Description

I have been exploring the medium is the message, and my whole idea for this project was based on the western-centric lamguage design of most programming languages and the beauty of esoteric languages. This language which for now has the stupid name emolang is a language which completely exists out of emojis, syntax is hard, numerical values are defined in binary even floating point numbers. For the proof of concept I wrote the code you can see on the left. It generates three stories made out of emojis, chosen from prompts in the code.
